Border-Radius Previewer

Como funciona o Border Radius?

O Border Radius é uma propriedade, em CSS, usada para arredondar bordas. Nela, temos quatro parâmetros, um para a borda superior esquerda, superior direita, inferior esquerda e inferior direita.

Coloque um número em cada border-radius. (Será convertido para pixels)

CSS Element:

border-radius: none
